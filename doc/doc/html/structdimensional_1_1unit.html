Detailed Description

template<int length, int mass, int time, int current, int temperature, int amount_of_substance, int luminous_intensity>
struct dimensional::unit< length, mass, time, current, temperature, amount_of_substance, luminous_intensity >

The unit struct is used to describe physical units.

This template use 7 international base units to describe all physical units, to guarantee strong-typed unit analysis.
Directly use this struct is not suggested, use unit_multiply, unit_divide, unit_power and unit_root with typedef/using to generate derived unit with exisiting units.
