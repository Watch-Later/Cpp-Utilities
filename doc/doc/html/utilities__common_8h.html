Detailed Description

Macros defined for utilities.

This file provides macros defined for all utilites.
If macro UTILITIES_NAMESPACE is defined to a specific symbol name, all utility symbols will be defined in this namespace.
Example:

#define UTILITIES_NAMESPACE utilities
